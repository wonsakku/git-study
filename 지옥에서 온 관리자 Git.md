## 섹션2. Git 중급  

<br>

### git reset 

___

|옵션|용도?|
|:--:|--|
|soft|커밋 로그 변경 시|
|mixed|작업 영역의 내용 변경 시|
|hard|변경 내용 완전 제거?|

<br><br>

### git reflog  

___

> git log history 전체 출력. reset --hard한 내용도 출력이 되기 때문에 reset --hard를 사용하면 reset --hard 이전으로 돌아갈 수 있다.  

<br><br>

### amend  

___

> git commit --amend -m "~~~~~"
